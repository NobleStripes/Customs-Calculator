Release date: April 30, 2026

## Highlights

- Added full PHL 2026 customs compliance fee and tax engine, covering all CMTA/CAO administrative fees, excise tax categories, corrected VAT base, BOC weekly FX rates, and all three entry tiers (de minimis / informal / formal).
- Added goods classification engine for Philippine import regulation: CMTA import type (Free / Regulated / Restricted / Prohibited), agency clearance requirements, strategic trade (STMO/RA 10697) flag, VAT-exempt goods (NIRC Sec. 109) flag, and FTA Certificate of Origin form identification.

## PHL 2026 Compliance Fees and Tax Engine

Added a comprehensive fee and excise computation layer aligned with current BOC/BIR issuances:

- **De minimis threshold** set to ₱10,000 (CMTA Sec. 423; EO 171 s.2022).
- **Entry tiers:** `de_minimis` (≤ ₱10,000 FOB), `informal` (₱10,001–₱50,000), `formal` (> ₱50,000).
- **Dutiable value:** FOB + insurance + freight (CIF basis per CMTA Sec. 201).
- **Administrative fees:** Import Processing Fee (IPF) on tiered FOB thresholds, Customs Documentary Stamp (CDS ₱265), BIR Documentary Stamp Tax (DST ₱30), Legal Research Fund (LRF ₱10), Arrastre/Wharfage, brokerage.
- **VAT base:** dutiable value + customs duty + surcharge + brokerage + arrastre/wharfage + Dox/Stamp/Other charges + all administrative fees, per NIRC Sec. 107 as amended by TRAIN.
- **Excise tax engine** (RA 10963 TRAIN / RA 11467): spirits, wines, beer, tobacco, automobiles (up to ₱4M threshold), sweetened beverages, petroleum products; both ad-valorem and specific rates.
- **BOC weekly FX rate:** live fetch from BOC memoranda page with 7-day TTL; priority chain: BOC weekly → live market API → market cache → hardcoded fallback. `fxPreferBocRate` runtime setting (default `true`).
- **Insurance benchmark:** 5% of (FOB + freight) applied when declared insurance is zero.

## Goods Classification Engine

New `importClassification` service classifies any 10-digit AHTN/HS code against Philippine regulatory frameworks:

- **Import type (CMTA Sec. 116–118):** `free` | `regulated` | `restricted` | `prohibited` — chapter-level defaults for all 99 HS chapters with heading-level overrides for 30+ sensitive headings.
- **Agency clearance:** returns abbreviated agency codes (e.g., FDA, FPA, NTC, SRA) and full agency names for all clearance requirements per heading.
- **Strategic trade flag:** identifies goods under RA 10697 (Strategic Trade Management Act); covers radioactive materials (2844), radar/navigation (8526), optical/laser equipment (9013), explosives (3601–3604), and other controlled items. `isStrategicTradeGood` + `strategicTradeNotes` exposed on result.
- **VAT-exempt flag:** identifies goods exempt under NIRC Sec. 109 including agricultural/marine staples, medicines, fertilizers, seeds, and books. `isVatExempt` + `vatExemptBasis` exposed on result.
- **FTA Certificate of Origin:** `requiresCertificateOfOrigin` set for non-MFN FTA schedule codes (ATIGA, ACFTA, AHKFTA, AJCEPA/JPEPA, AIFTA, AANZFTA, RCEP, ACDFTA, PH-EFTA). Form identifiers returned: Form D, Form E, JPEPA CoO, RCEP CoO, Form AI, Form AANZ, RCEP CoO, Form PH-EFTA.
- **Warnings:** EO 156 vehicle import restrictions (8703), Dangerous Drugs Board (2939/3002), CITES (0106/4101/0407), strategic goods advisory — surfaced as a `warnings[]` array on the result.
