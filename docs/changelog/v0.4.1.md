# v0.4.1

Release date: April 29, 2026

## Highlights

- Added CSV/XLS/XLSX upload support to the Admin tariff import workspace.
- Added Admin export actions for tariff import preview results and tariff source governance tables.
- Extended the tariff preview/import API contract so workbook uploads can be parsed server-side through the existing tabular ingestion flow.
- Expanded tariff HTML parser coverage with Tariff Commission matrix fixtures and BOC memorandum negative fixtures based on real source page shapes.
- Fixed Batch Import CSV parsing so reordered headers and common alias names are mapped correctly.
- Bumped package version to `0.4.1`.

## Upgrade Notes (Operators)

- The Tariff Sources tab now accepts pasted CSV, `.csv` uploads, and `.xls`/`.xlsx` workbook uploads for tariff staging.
- Preview and source-governance tables can now be exported directly as CSV from the Admin UI.
- Workbook uploads are normalized through the same tariff import pipeline used by CSV imports.
